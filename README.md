# Letras Vivas · Живые буквы · Living Letters

**An interactive poetry animator — and a small empathy machine.**

For most readers, letters sit still on the page. For many people with dyslexia, they don't: characters seem to flip, mirror, swim, tremble, or swap places. *Living Letters* turns any poem into a living, moving text so that a fluent reader can experience — for a moment — what an unstable page might feel like, and so that this experience can be shared as art.

🔗 **Try it:** `https://webuela.github.io/dancing_letters/` *(choose your language: ES / RU / EN)*

## What it does

Paste any poem, pick a scene, press **Animate**:

- **14 animation scenes** — flip, swing, bounce, wave, chaos, breathe, spiral, vortex, drum, pulse and more
- **Dyslexia-inspired effects** — mirrored letters, letter **swapping** (b↔d, p↔q; б↔д, п↔н for Cyrillic), trembling, drifting, blinking
- **Secret word** — hide a word inside the poem, highlighted letter by letter
- **Full styling** — background image or color, text color, darken overlay, size, speed, intensity
- **Projects** — save, load, export and import your compositions (stored locally in your browser)
- **Presentation mode** — full-screen playlist of your saved poems
- **Export** — record the animation as video (Chrome/Edge) or download every frame as a PNG sequence (ZIP) for video editing

## Languages

The interface and sample poems are available in **Español**, **Русский** and **English** — switch anytime from the sidebar.

## Tech

One self-contained HTML file per language. No frameworks, no build step, no server, no tracking. Fonts and icons load from public CDNs; everything else runs entirely in your browser.

## A note on dyslexia

This project is an **artistic simulation, not a medical or diagnostic tool**. Dyslexia is different for every person, and no animation can capture it exactly. The goal is empathy and curiosity: if these dancing letters make reading feel effortful for a minute, imagine navigating every page that way — and how much attention, patience and clever strategy that takes. If you want to learn more, look up your local dyslexia association.

## Running locally

Download the repository and open `index.html` in any modern browser. That's it.
